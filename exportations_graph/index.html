Graph Visualization

Welcome

Welcome to our neutrality project webpage.
Our goal is to try to detect network neutrality violations with traceroute measurements.
We created a Visualization application to show Graphs with these neutrality violations.
On this web page we will present some functions of our Graph Visualization Application.

Cluster Vertices

To avoid displaying all vertices at the same time, they are clustered to show only useful data.

Link colors

A link which is suspected to be non-neutral is displayed in red, otherwise it is displayed in green.

Clicking on router

Clicking on a router show all links related to it.

Mouse on router

Positioning the mouse on a vertex shows its IP.

Mouse on link

Once an edge is displayed, positioning the mouse on a link shows useful information (source IP, destination IP, neutral value).

Clicking on edge

Clicking on an edge shows all paths passing through this edge.

Vertices Option

Using the right top menu allows to choose if routers must be displayed or not.

Overlapping routers

If there is routers overlapping (same position), they are disposed in spiral and can be used in the same way as other vertices.
